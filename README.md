# Plant-Disease-Detection
This project implements a deep learning-based plant disease detection system using EfficientNet, a state-of-the-art convolutional neural network architecture. The model is trained to classify plant leaf images into healthy and diseased categories with high accuracy.

The system is designed to assist farmers and researchers in identifying plant diseases early, enabling timely intervention and improved crop yield.

* Features
Detects plant diseases from leaf images
High accuracy using EfficientNet
Supports multiple disease classes
Can be deployed on cloud or local system
Extendable to mobile applications (TensorFlow Lite)

**Model Used
This project uses EfficientNetB0, part of the EfficientNet family, known for:
Better performance with fewer parameters
Compound scaling (depth, width, resolution)
Faster training and inference

 **Dataset
The model is trained on the PlantVillage Dataset, which contains:
Multiple plant species (Potato, Tomato, etc.)
Healthy and diseased leaf images
Labeled data for supervised learning
